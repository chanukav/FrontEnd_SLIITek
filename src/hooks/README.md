# Custom Hooks

This directory contains custom React hooks to encapsulate reusable logic.

## Purpose

- Extract complex logic from components to keep them clean.
- Share stateful logic between multiple components.
- Handle side effects, API calls, or subscriptions in a reusable way.

## Examples

- `useFetch.js`: A hook to handle data fetching.
- `useForm.js`: A hook to manage form state and validation.
- `useWindowSize.js`: A hook to track window dimensions.
